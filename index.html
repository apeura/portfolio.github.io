Learn English!

A full stack application meant for studying English - Finnish word pairs. User can choose one or more categories at a time (or get a random table through "Surprise me!"), the tables pull a random set of words fitting the theme from the database. After submitting their answers the user can see their score on said table. Includes an admin page where word pairs can be added, edited and deleted.

Note: The database was hosted by TAMK and has since been closed, table data is no longer displayed.

Weather App

A mobile app that shows weather forecasts utilizing OpenWeather API. The app displays a detailed daily forecast including but not limited to current temperature, current "feels like" temperature, highest and lowest temperature of the day, upcoming forecast spaced in 3 hour intervals and upcoming week's forecast. The app also includes a dark and light theme.

# Portfolio
A mobile-first portfolio page designed to showcase projects and other relevant information. 

## Features
- **Mobile-First Design**: Optimized for mobile devices with a responsive layout.
- **Custom Visuals**: Self-made graphics for a personalized touch.
- **Technologies Used**:
  - **HTML** & **CSS** for structure and styling.
  - **Font Awesome Icons** for clean, scalable vector icons.
  - **Figma** for wireframing and design prior to implementation.
